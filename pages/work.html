Centipede Lite Editable

Showdemo using C++ and the Allegro5 development framework, to demonstrate the editing capabilities that can be achieved with ImGui. To store the data, sQLite3 is used.

Two Joysticks

Small demo made with Unity Engine and C#, where the challenge was to create a game where an exclusive use of the analog sticks was made. The player has to find the exit, having in his favor, that he can move the walls, changing the structure of the area.

Run!! Dodge!! Trivia!!

Mobile game made in Unity and C#. Recreation of a Danganronpa mini-game, where you have to answer questions from different categories while dodging obstacles. Active use is made of Open Trivia Database to reload the question rounds. Specific mobile features such as the accelerometer are taken as another input option.

City Generator

A small city generator made with a partner using C++, where the main emphasis is on the implementation of a real-time editor using ImGui, and shader techniques for the different illuminations in the scene using OpenGL and GLSL.

//-//

Agent's Pattern

Demo made in C++ and SDL2 to demonstrate the basic movement patterns that can be implemented in artificial intelligence agents.

//-//

Dungeon Generator

Demo made in Unity Engine and C#, using the Cellular Automata, Marching Squares, and Flood Fill algorithms to generate a cave system. All zones are connected and a path is generated given two random points using the A* Pathfinding algorithm.

//-//

Agent's Perception

Short demo using C++ and Unreal Engine, to demonstrate the perceptions that artificial intelligence agents can have. Agents have to find the key to get out of the maze, and watch out for the warden. The top down approach with behavior trees is present.

//-//

Landscape Effect

A small demo using C and SDL. Made with another partner, we recreated a landscape using height maps, and optimized the code to run on a Raspberry 3.

//-//

Metaballs 2D

A small demo using C and SDL, to recreate the Metaballs Effect. Made in conjunction with another partner, the code is optimized to run on a Raspberry 3.

Hideout. Face your Fears

Final year project at ESAT using C++ and Unreal Engine. Third-person stealth game where the player will have to overcome numerous dangers in a post-apocalyptic world. My contribution to the project was prototyping the artificial intelligence, integration with Steam Store and its achievements, and general programming.
